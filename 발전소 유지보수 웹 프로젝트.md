# 🏭 발전소 유지보수 웹 프로젝트 (PowerPlant-Maint-Web)

## 1. 프로젝트 개요 (Overview)
- **목표:** 오프라인 환경에서 서버, 네트워크 장비, 문서를 통합 관리.
- **환경:** 폐쇄망 (No Internet), Docker 기반 배포.
- **기술 스택:** Python (FastAPI/Django), React/Vue, PostgreSQL/SQLite.

## 2. 데이터베이스 스키마 (Database Schema)
### Table: Systems (설비 구성)
*계층형 구조: 1CC > GT, ST...*
- **id:** (PK)
- **name:** 시스템명 (예: 1CC, DCS, System1)
- **parent_id:** (FK) 상위 시스템 ID (NULL이면 최상위)
- **description:** 설명

### Table: Locations (설치 위치)
*계층형 구조: 1block > 전산실...*
- **id:** (PK)
- **name:** 위치명 (예: 1block, 중앙제어실)
- **parent_id:** (FK) 상위 위치 ID (NULL이면 최상위)
- **description:** 설명
